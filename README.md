# 🚗 EVOA Seminovos

Projeto desenvolvido para fins de estudo e aprimoramento das minhas habilidades em desenvolvimento Web (Front-end e Back-end).

O objetivo é criar um site moderno para uma concessionária fictícia de veículos seminovos, utilizando HTML, CSS, JavaScript e PHP, sem banco de dados.

---

## 📚 Tecnologias

### Front-end

* HTML5
* CSS3
* JavaScript

### Back-end

* PHP 8+

## 🎯 Funcionalidades

### Front-end

* Página inicial moderna
* Catálogo de veículos
* Página individual do veículo
* Layout responsivo
* Menu responsivo
* Carrossel de imagens
* Galeria de fotos
* Modo escuro
* Botão "Voltar ao topo"
* Animações de interface

### Back-end

* Componentização com PHP (`header`, `navbar` e `footer`)
* Listagem dinâmica de veículos
* Sistema de pesquisa
* Filtros por marca, ano e preço
* Página dinâmica de cada veículo
* Simulação de financiamento
* Funções reutilizáveis
* Organização de dados em arquivos PHP

---

## 📌 Objetivo

Este projeto foi desenvolvido exclusivamente para estudos e para compor meu portfólio no GitHub.

Ele não possui integração com banco de dados nem representa uma concessionária real.

Seu principal objetivo é praticar a criação de aplicações web organizadas utilizando HTML, CSS, JavaScript e PHP.
